A blockchain-powered art marketplace with integrated AI art generation tools and TOLA token functionality.

== Description ==

VORTEX AI Marketplace is a comprehensive solution for creating, showcasing, and selling AI-generated artwork. The platform integrates cutting-edge AI models with blockchain technology to provide a secure, transparent marketplace for digital artists and collectors.

= Key Features =

* **AI Art Generation**: Create stunning artwork using state-of-the-art AI models like Stable Diffusion, DALL-E, and Midjourney
* **HURAII AI System**: Our proprietary AI system designed specifically for artists
* **Blockchain Integration**: Secure transactions and ownership verification
* **TOLA Token**: Native utility token for marketplace transactions
* **Artist Profiles**: Custom profiles for artists to showcase their work
* **Comprehensive Analytics**: Detailed insights for artists and administrators
* **Ranking System**: Promote top artists and trending artworks
* **Multilingual Support**: Available in multiple languages

= For Artists =

* Create and sell AI-generated artwork
* Build a customizable profile
* Track sales and performance metrics
* Receive payment in cryptocurrency or traditional currency
* Join a community of digital artists

= For Collectors =

* Discover unique AI-generated artwork

= For Administrators =

* Content moderation tools
* Commission management
* User management

== Installation ==

1. Upload the `vortex-ai-marketplace` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin settings through the 'VORTEX' menu in your admin dashboard
4. Set up your blockchain connections and API keys as needed
5. Begin creating and listing artwork!

= Minimum Requirements =

* WordPress 5.8 or higher
* PHP 7.4 or higher
* MySQL 5.6 or higher
* HTTPS enabled site for secure transactions

== Frequently Asked Questions ==

= Is blockchain integration required? =

While the VORTEX AI Marketplace is designed to work best with blockchain integration for secure transactions and ownership verification, you can use the basic marketplace functionality without it. However, TOLA token features will require blockchain integration.

= Which blockchain networks are supported? =

VORTEX primarily supports Solana for its efficiency and low transaction costs. Support for additional networks may be added in future updates.

= Do I need API keys for the AI models? =

Yes, to use the AI generation features, you'll need to obtain API keys for the specific models you want to use (Stability AI, OpenAI, etc.). These can be configured in the plugin settings.

= Can I use my own AI models? =

Advanced users can integrate custom AI models through our extension system. Documentation for this feature is available in the developer documentation.

= Is the plugin compatible with WooCommerce? =

Yes, VORTEX AI Marketplace can integrate with WooCommerce for additional e-commerce functionality.

= How are transactions processed? =

Transactions can be processed through traditional payment gateways or using cryptocurrency through blockchain integration, depending on your configuration.

= How do I set artist commissions? =

Administrator can set platform-wide commission rates, and individual artists can set their own pricing for their artwork.
