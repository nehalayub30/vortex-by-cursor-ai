VORTEX AI Marketplace is a blockchain-powered platform for AI-generated art with built-in HURAII AI generator and comprehensive metrics system.

== Description ==

VORTEX AI Marketplace transforms your WordPress site into a complete blockchain-powered marketplace for AI-generated art. This comprehensive plugin integrates HURAII AI generation tools, blockchain transactions with TOLA tokens, and a full suite of analytics and metrics.

= Key Features =

* **Complete Marketplace** - Browse, buy, and sell AI-generated artwork
* **HURAII AI Generator** - Integrated AI tools for creating artwork:
  * Text-to-image generation
  * Image-to-image transformation
  * Style transfer capabilities
  * Advanced image enhancement
* **Blockchain Integration** - Secure transactions with TOLA tokens:
  * Smart contract implementation
  * Automatic royalties for artists
  * Blockchain verification for authenticity
* **Comprehensive Metrics** - Track performance with real-time analytics:
  * Artist ranking systems
  * Artwork popularity metrics
  * Sales and engagement analytics
  * Marketplace trends
* **Multilanguage Support** - Reach a global audience:
  * 8+ core languages included
  * RTL language support
  * Full translation capability

= Who Is This For? =

* **Artists** - Showcase and sell AI-generated artwork
* **Collectors** - Discover and purchase unique digital art
* **Galleries** - Create an online presence with blockchain verification
* **Art Communities** - Build specialized marketplaces for AI art

== Installation ==

1. Upload the `vortex-ai-marketplace` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin settings via the 'VORTEX' menu item
4. Set up your marketplace appearance and functionality
5. Configure HURAII AI integration and blockchain settings
